## Описание программы

LiveControl - программа, предназначенная для автоматического запуска приложений и контроля их работоспособности. При падении или зависании контролируемой программы, она будет автоматически перезапущена.
Приложение позволяет автоматически запускать любые программы и перезапускать их при падении. Для контроля зависаний требуется добавление в контролируемую программу API (github:http...).    

## Основные особенности

- **Автозапуск при старте ОС**: Приложение стартует при запуске ОС и автоматически запускает контролируемые приложения
- **Старт нескольких приложений**: Позволяет запустить сразу несколько приложений одним кликом
- **Перезапуск при падении**: При обнаружении падения контролируемой программы, она будет автоматически перезапущена 
- **Контроль выполнения**: Защита от случайного закрытия контролируемой программы
- **Перезапуск при зависании**: При интеграции API в контролируемое приложение, LiveControl получает возможность обнаруживать зависание программы
- **Список программ онлайн**: Программа имеет встроенный сервер, подключившись к которому можно удалённо получить данные о состоянии программ

## Системные требования

- **ОС**: Windows 10/11

## Работа с программой

1. Перейдите в директорию настроек программы, используя меню File, Settings 
2. Используйте extern(child).txt для запуска дочерних приложений. Эти приложения будут автоматически закрыты при закрытии LiveControl. Так же для них есть возможность разрешить или запретить закрытие приложения пользователем.  
3. Используйте extern(pid).txt для контроля приложений, используя список процессов Windows. При отсутствии программы в списке запущенных приложений, эта программа будет запущена LiveControl.
4. При использовании API LiveControl вашей программой, вы самостоятельно можете изменить TCP порт и активность сервера LiveControl. 
5. Для удалённого доступа к состоянию и списку контролируемых программ, отредактируйте секцию [HTTP] в настройках программы.
6. Для включения/выключения автоматического запуска LiveControl при старте системы, запустите программу с правами администратора.
